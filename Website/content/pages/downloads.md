## Package Managers

### DnsClientX for .NET

- Package: [DnsClientX on NuGet](https://www.nuget.org/packages/DnsClientX)
- Releases: [GitHub Releases](https://github.com/EvotecIT/DnsClientX/releases)

```bash
dotnet add package DnsClientX
```

### DnsClientX for PowerShell

- Module: [DnsClientX on PowerShell Gallery](https://www.powershellgallery.com/packages/DnsClientX)
- Examples: [Module example scripts](https://github.com/EvotecIT/DnsClientX/tree/main/Module/Examples)

```powershell
Install-Module DnsClientX
```
